# esp-list

A list of Email Service Providers.

## Installation

```sh
npm install esp-list
```

## Usage

```js
const esps = require('esp-list')
```

`esps` will be a JSON object with the following structure:

```js
[
  {
    name: "ESP name",
    url: "https://www.example.com/"
  },
  {
    // ...
  }
]
```
